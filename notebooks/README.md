# Notebooks

Hand-crafted walk-throughs that layer on top of the OpenPathAI
library. Every notebook is self-contained and runs without downloading
any dataset (synthetic tile generators from Phase 3 are used).

| Notebook | Phases exercised | Platform |
|---|---|---|
| [`01_quick_start.ipynb`](01_quick_start.ipynb) | Data layer (2), training (3), explainability (4), CLI (5) | CPU (laptop, Colab) |

Other notebooks ship with later phases:

* `04_colab_reference.ipynb` — hand-crafted reference for the Colab
  exporter (Phase 11).
* `05_foundation_models.ipynb` — Tier-C feature extraction (Phase 13).
* `07_zero_shot.ipynb` — CONCH + MedGemma pipelines (Phase 15).
